This project is built using React, TypeScript, TanStack Router, TanStack DB, TanStack Query and Tailwind CSS. It uses the Open-Meteo API to pull various weather Information based on location search.

## Features

- Smart Search: Location autocomplete with favorites support
- Metric and Imperial Unit support
- Theme (light / dark mode) support
- Responsive Design: Works on desktop, tablet, and mobile devices
- Real-time Weather: Current conditions, hourly forecast, and 10-day outlook
- Detailed Metrics: Temperature, wind, sunrise / sunset and more
